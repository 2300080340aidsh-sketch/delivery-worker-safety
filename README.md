# Delivery Worker Safety – AI Powered Insurance Platform

## Project Overview
This project focuses on building an AI-powered insurance platform for delivery workers in India. Many gig workers lose income due to external disruptions such as heavy rain, floods, pollution, or sudden shutdowns. Our platform provides weekly income protection and automatic claim payouts.

## Problem Statement
Delivery partners working in food delivery and e-commerce often face income loss when environmental or social disruptions stop their work. Currently, they do not have proper insurance protection for such situations.

## Our Solution
We are building a parametric insurance platform that:
- Provides weekly insurance plans
- Detects disruptions using weather and location data
- Automatically triggers claims
- Gives instant payouts for income loss
- Uses AI for risk analysis and fraud detection

## Target Users
Delivery workers from platforms such as:
- Food delivery
- E-commerce delivery
- Grocery delivery

## Key Features
- AI-based risk assessment
- Weekly insurance policy
- Automatic disruption detection
- Auto claim processing
- Fast payout system
- Worker dashboard for tracking earnings and claims

## Weekly Premium Model
The platform offers affordable weekly plans.  
Example:
Basic Plan – ₹49 per week  
Pro Plan – Higher coverage for workers with higher risk zones.

## Parametric Triggers
Claims are automatically triggered when:
- Heavy rain
- Flood alerts
- Extreme pollution levels
- Sudden curfews or disruptions

## AI/ML Integration
AI will be used for:
- Risk prediction
- Dynamic premium calculation
- Fraud detection
- Data analysis based on worker location and activity

## Tech Stack
- Frontend: Mobile App UI (Figma Prototype)
- Backend: Node.js / Python (Planned)
- Database: Firebase / MongoDB
- APIs: Weather API, Location API
- Payment Integration: Razorpay / UPI (Simulation)
